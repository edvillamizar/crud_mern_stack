# CRUD MERN Stack

Aplicación web fullstack para gestión de registros desarrollada con el stack MERN.
Incluye backend con API REST y frontend conectado al servidor.

## 🛠️ Tecnologías

- **MongoDB** — base de datos NoSQL
- **Express.js** — servidor y rutas de la API
- **React** — interfaz de usuario (cliente)
- **Node.js** — entorno de ejecución del servidor

## ⚙️ Funcionalidades

- Crear, leer, actualizar y eliminar registros (CRUD completo)
- API REST con rutas separadas por recurso
- Conexión a base de datos MongoDB
- Frontend en React consumiendo la API

## ▶️ Cómo ejecutarlo localmente

1. Clona el repositorio:
```bash
   git clone https://github.com/edvillamizar/crud_mern_stack.git
```
2. Instala dependencias del servidor:
```bash
   npm install
```
3. Instala dependencias del cliente:
```bash
   cd cliente && npm install
```
4. Configura tu conexión a MongoDB en `conexion.js`
5. Inicia el servidor:
```bash
   node server.js
```
6. Inicia el cliente:
```bash
   cd cliente && npm start
```

## 📌 Contexto

Proyecto desarrollado como ejercicio práctico para aprender
el stack MERN y la integración entre frontend React y backend Node/Express.
